#### Local setup

To run this project locally, follow these steps.

1. Clone the project locally, change into the directory, and install the dependencies:

```sh
git clone https://github.com/hominids/app.git

cd app

# install using NPM or Yarn
npm install

# or

yarn
```

2. Start the app

```
npm run dev

# or

yarn dev
```
